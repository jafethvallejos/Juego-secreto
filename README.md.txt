# Juego Secreto

Este proyecto es parte del curso de Alura Latam (programa ONE) y Oracle Next Education.  
La aplicación permite a los usuarios ingresar nombres de amigos en una lista, visualizarla, y realizar un sorteo aleatorio para determinar quién es el “Amigo Secreto”.

## Funcionalidades

- Agregar nombres a la lista de amigos.
- Validar que el campo de entrada no esté vacío ni se repitan nombres.
- Visualizar la lista de amigos ingresados.
- Sortear un amigo secreto aleatoriamente y mostrar el resultado en pantalla.

## Tecnologías utilizadas

- HTML
- CSS
- JavaScript
